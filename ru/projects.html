Ключевой проект

Оптимизация запасов и поддержка принятия решений

Разработка framework для оптимизации политики пополнения запасов с учетом риска дефицита, затрат на хранение, service-level targets, lead times, минимальных партий заказа и складских ограничений.

  • Разработана логика пополнения и целевых уровней запасов для портфеля номенклатуры.
  • Рассчитаны safety stock и target stock параметры в увязке с KPI надежности.
  • Реализована сегментация спроса для регулярных, нерегулярных и сезонных паттернов.
  • Построен decision-support framework для практических решений по пополнению в условиях реальных ограничений.

Дополнительные проекты

Оптимизация производственного workflow

Разработка оптимизационных моделей для последовательно-параллельных производственных процессов с целью повышения сквозной эффективности обработки заказов.

  • Применена логика greedy algorithms для упорядочивания производственных операций.
  • Фокус сделан на практическом росте пропускной способности и согласованности процессов.
  • Снижение сквозного lead time обработки заказов на 50% за счет оптимизации последовательности операций в sequential-parallel workflows.

Оптимизация складского footprint

Оптимизация размещения и консолидации складов с учетом ограничений по площади и операционных требований.

  • Разработана логика консолидации для сокращения складского footprint.
  • Снижены затраты на хранение и содержание складских запасов на 30%.
  • Решения по консолидации складов увязаны с политикой запасов, что позволило сократить warehouse holding costs на 30%.

Python-инструмент для scenario analysis

Разработка аналитического инструмента на Python для сценарного анализа, sensitivity testing и рекомендаций по параметрам пополнения.

  • Поддержана работа со сценарным моделированием и анализом чувствительности.
  • Сопоставлены trade-offs между затратами, риском и service-level decisions.
  • Поддержка decision-makers через сценарное сравнение, sensitivity testing и cost-risk trade-off analysis.

Тепловое моделирование железнодорожной насыпи

Тепловое моделирование грунтов основания железнодорожной насыпи при изменении условий снежного покрова.

  • Исследовано тепловое поведение оснований инфраструктуры при различных условиях снежного покрова.
  • Инженерный анализ связан с математически структурированным процессным моделированием.
  • Проект связан с публикацией 2026 года о деградации вечномерзлых грунтов основания железнодорожной насыпи при вариации высоты снежного покрова.
